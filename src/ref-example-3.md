## 步骤3

创建测试元素并添加transform属性

```html
<style type="text/css">
.greenSquare {
    position: absolute;
    top: 0;
    left: 0;
    width: 100px;
    height: 100px;
    background: green;

    transform: translate(100px, 100px);
}
</style>
```

```html
<body>
    <div class="greenSquare"></div>
</body>
```
